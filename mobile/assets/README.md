# Assets Folder

Folder ini berisi assets untuk aplikasi (icon, splash screen, dll).

## Required Assets

Untuk menjalankan aplikasi, Anda perlu menambahkan:

1. **icon.png** - App icon (1024x1024)
2. **splash.png** - Splash screen (1242x2436)
3. **adaptive-icon.png** - Android adaptive icon (1024x1024)
4. **favicon.png** - Web favicon (48x48)

## Generate Assets

Anda bisa generate assets menggunakan:
- [Expo Asset Generator](https://www.npmjs.com/package/expo-asset-generator)
- Atau buat manual sesuai ukuran di atas

## Temporary Solution

Untuk development, Expo akan menggunakan default assets jika file tidak ditemukan.
